# agent-ecosystem-contracts

Shared Python contracts for event and state objects across multi-agent tooling.

## Included Contracts

- `Provider`
- `Message`
- `ToolCall`
- `ToolResult`
- `Session`
- `Task`

These contracts are intended to be lightweight, dependency-free, and stable
enough to serve as an interoperability layer between orchestration runtimes,
analytics tools, and transcript processors.
